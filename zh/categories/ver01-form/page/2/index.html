JsonSchema

@ver01/form extended Json Schema define with $vf_opt section

$vf_opt.props

will mixin to all components belone to one value node. Filtered by Component Schema propsMixinList

$vf_opt.widget

string, assign the component used. Defined in Component Schema widgets

$vf_opt.option

  • For array
    • orderable(default true)
    • addable(default true)
    • removeable(default true)
  • For string
    • disabled
    • readonly
  • For object
    • order: [‘first-key’, ‘second-key’, ‘*’, ‘last-key’]

$vf_opt.validate

[ruleName]: ruleValue

see sample-validation local demo

Validate can be defined in formProps or buildin validate
